About Menu Buttons

Simply put, it’s a collection of menu buttons developed with HTML, SVG, and CSS. You can use them as is, or (better yet) adapt them to your own project.

One thing to note: Internet Explorer (versions ≤ 11) does not support transitions on SVG elements in the HTML DOM. In many of those cases, you can replace these shapes with div elements instead.

Another thing to note: because these are demos, and not actual links to menus, I'm using button elements. Semantically-speaking, a elements with an internal href value (e.g. <a href="#menu">) may be more appropriate.
